modules in python is simply a file consisting of python code

The extension of file is either .py or .pyw

why to use modules:
Modules `help in grouping of related code (Functions or Classes) into a module which makes the code easier to use and understand

examples of importing modules
------------------------------

import random
import time
OR
import random,time, os, sys

To import a function or a class from a module

from random import randint
from random import random, choice, randint
